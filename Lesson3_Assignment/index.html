Highlight #1 What is Git?

Git is knows as a version control system. This means that it records changes that you have made to files over time. This is helpful because you can save a file and create a seperate file. Once the seperate file has been created, you can make as many changes as you want to it and not have to worry about it impacting you other files. Git is used because many projects can be worked on at once and you can make as many changes as you want but if a mistake is made, there is no serious consequence.

Highlight #2 Git Commands

The commands in Git are extremely important. They allow users to split the project into multiple parts, undo actions they have made, combine different branches, and even send your projects to different servers.

Highlight #3 Git Repositories

To make a Git repository, users can either take the repository and add it to their computer, or users can clone the projects file and use the clone on their computer. To add the repository to a directory, users create a sub-branch of the file and then add it to their computers. To clone a repository, users must get the URL of the file and the duplicate it within Github.
